How to Set Up a Web Server With Docker


Step 1: Pull the Ubuntu image
Ex: docker pull ubuntu

Step 2: Run the Ubuntu image and bash shell interactively
Ex: docker run -it ubuntu:latest /bin/bash

Step 3: Update the image system using the container bash shell
Ex: apt-get update

Step 4: Install apache using the container bash shell
Ex: apt-get install apache2

Step 5: Install vim text editor
Ex: apt-get install vim

Step 6: exit Ubuntu container
Ex: exit

Step 7: Commit container to create a new Docker image
Ex: docker commit container_id ubuntu_webserver

Step 8: Share data between the docker container and host
Ex: docker run -it -p 127.0.0.1:80:80 -v C:\Docker_tutorial\local_dir:/var/www/html ubuntu_webserver /bin/bash

Step 9: Change directory to /var/www/html/ and create a index.html
Ex: cd /var/www/html/ vim index.html

Step 10: Edit index.html to create your homepage
Step 11: Go to your host browser and type 127.0.0.1 in the url to see your homepage
