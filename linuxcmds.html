Basic Linux Commands


man - manual pages/help
pwd - print working directory
mkdir - make directory
cd - change directory
ls - list files
cp - copy file(s)/folder
mv - move/rename files/folders
rm - delete files/folders
less - list file contents...in a pager
cat - catenate (concatenate) - reads one or more files and prints them to standard output
