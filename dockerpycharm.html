How to connect Docker with Pycharm


Step 1: Open a project on Pycharm

Step 2: On top right, click file then go to settings

Step 3: Under Build, Execution, and Deployment click on Docker.

Step 4: On Connect to Docker daemon it should be default on Docker for Windows.

Step 5: Wait a bit then it should say connection successful. Click Ok.

Step 6: On top right, click Add Configuration.

Step 7: Click the + sign. Under Docker click on Dockerfile.

Step 8: Create a name, find the path of Dockerfile, and you can add a name for the image tag. Run built image should be checked by default. Click Apply

Step 9: On the bottom right, click on No interpreter then click add interpreter

Step 10: Click on Docker and click the dropdown and find the image name you created.
